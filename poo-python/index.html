La programmation orientée objet en Python

La programmation orientée objet en Python

  • Structurer les données sous formes de classes
  • Contient des attributs et des méthodes
  • Est construit à l'aide d'un constructeur (__init__)

Vocabulaire

Classe : Moule définissant la manière de créer un objet. Contient des attributs et des méthodes

Objet : Instance d'une classe. Contient des attributs propres à cette instance

Attributs : Caractéristiques d'un objet

Méthodes : Fonctions permettant la manipulation d'un objet et de ses attributs

Constructeur : Méthode expliquant comment créer un objet (valeurs par défaut des paramètres, ...)

Self : Mot-clef utilisé pour parler de l'instance courante de la classe

Exemple

                        
                            class Person:

                                def __init__(
                                    self,
                                    nom = 'Doe',
                                    prenom = 'John',
                                    age = 30
                                ):
                                    self.nom = nom
                                    self.prenom = prenom
                                    self.age = age


                                def greetings(self, other_person):
                                    print(f'Bonjour {other_person.prenom}')

Exemple

                        
                            person = Person()

                            print(f'{person.prenom} {person.nom}') # John Doe

                            person1 = Person(nom = 'Hudson', prenom = 'Saul', age = 59)

                            print(f'{person1.prenom} {person1.nom}') # Saul Hudson

Les méthodes magiques

  • Sont interprétées directement par Python (aucun besoin de les appeler directement)
  • Sont toujours encadrées de 2 underscores '__'
  • Permettent de modifier le comportement d'une classe dans certains cas
  • Servent à créer du sucre syntaxique

Les méthodes magiques

Exemples de méthodes magiques :

  • __str__ : Permet de modifier le résultat de sortie de la fonction str()
  • __init__ : Permet de créer une instance d'une classe
  • __len__ : Permet d'utiliser la fonction len() sur l'objet
  • __lt__ : Permet de comparer deux objets en utilisant le signe "<"
  • __gt__ : Permet de comparer deux objets en utilisant le signe ">"

Les méthodes magiques

                            
                                class Person:

                                    def __init__(self, nom, prenom, age):
                                        self.nom = nom
                                        self.prenom = prenom
                                        self.age = age

                                    def __str__(self):
                                        return f'{self.prenom} {self.nom}'

                                    def __len__(self):
                                        return self.age

                                    def __lt__(self, person):
                                        return self.age < person.age

                                    def __gt__(self, person):
                                        return self.age > person.age
                            
                        
                            
                                person = Person(nom = 'Hudson', prenom = 'Saul', age = 59)
                                person1 = Person(nom = 'Brodén', prenom = 'Joakim', age = 44)

                                print(f'La personne 1 s\'appelle {person}') # aka str(person)
                                # Output : La personne 1 s'appelle Saul Hudson

                                # aka : person1.__len__()
                                print(f'{person1} a {len(person1)} ans') # Joakim Brodén a 44 ans

                                if person < person1: # aka : person.__lt__(person1)
                                    print(f'{person1} est plus agée que {person}')
                                elif person > person1: # aka : person.__gt__(person1)
                                    print(f'{person} est plus agée que {person1}')
